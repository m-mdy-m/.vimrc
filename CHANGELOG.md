## [1.0.0] - 2025-12-22

### Added
- Initial release of VEX 
- Core vim configuration with sensible defaults
- UI theme with purple and gray color palette
- Custom netrw configuration for file browsing
- VEX start screen with ASCII logo and quick commands
- CLI tool `vex` for command-line operations
- LSP support via vim-lsp and vim-lsp-settings
- Language server configurations for TypeScript, Go
- FZF integration for fuzzy file finding
- status line with vim-airline
- Git integration with vim-gitgutter
- Comprehensive keyboard shortcuts following standard editor conventions
- Installation scripts 
- Plugin management with vim-plug
- Persistent undo history
- EditorConfig support
- Complete documentation suite
- Configuration documentation (keymaps, plugins, settings)
- Installation guides for Linux
- Health check command `vex doctor`
- Plugin management commands
- LSP management commands
- Version management system
